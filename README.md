Ce dépôt contient un projet web de gestion des invitations pour événements.  
Il permet de lister les invités par table avec un suivi dynamique de présence.  
Chaque invité peut être marqué comme présent via une icône d’approbation.  
Les compteurs "invités", "présents" et "absents" sont mis à jour en temps réel.  
Le projet utilise HTML, CSS et JavaScript pour l’interactivité.  
Design responsive pour une bonne adaptation aux petits écrans.  
Le code est clair, bien structuré et facilement personnalisable.  
Utile pour mariages, conférences, ou tout événement avec liste d’invités.  
Possibilité d’ajouter plus de fonctionnalités via JavaScript.  
Projet simple mais efficace pour apprendre à manipuler le DOM.
